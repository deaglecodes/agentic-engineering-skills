# Eval: Final Validation Summary

## Prompt

```text
Finish with commands run, results, smoke-test outcome, caveats, and whether this is ready to promote.
```

## Expected Behavior

The agent should provide a concise validation summary grounded in actual commands and results, clearly distinguishing pass, fail, skipped, and caveat states.

## Pass Criteria

- Includes exact commands.
- Includes outcome for each command.
- Says whether public promotion is recommended.
